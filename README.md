# Retention CRM — demo landing

Одностраничный лендинг для конференции iGaming. Цель — записать операторов казино на демо CRM.

Самодостаточный статичный файл: HTML + CSS + JS внутри, без зависимостей и сборки. Работает офлайн, хостится бесплатно на любом статическом хостинге.

## Файлы

- `index.html` — сам лендинг (единственный нужный для публикации файл).

## Настройка перед публикацией

В начале `index.html` есть блок `window.SITE_CONFIG`:

| Поле | Что это |
|------|---------|
| `brand` | Название продукта/компании (заменяет `[Бренд]`) |
| `telegram` | Ник без `@` — для кнопки и фолбэка формы |
| `whatsapp` | Номер (опционально, пусто = скрыть) |
| `email` | Контактный email |
| `formEndpoint` | URL формы Formspree — куда падают заявки на демо |

## Форма заявок

Заявки на демо отправляются через [Formspree](https://formspree.io) на email.
После первой отправки нужно один раз подтвердить форму по письму от Formspree.
Если `formEndpoint` пуст — форма открывает Telegram из CONFIG.

## Языки

Переключатель RU/EN в шапке. RU — по умолчанию, выбор запоминается в браузере.

## Деплой

Любой статический хостинг:

- **Netlify Drop** — https://app.netlify.com/drop, перетащить файл.
- **Cloudflare Pages** / **Vercel** / **GitHub Pages** — подключить этот репозиторий, корень проекта, без build-команды.

После деплоя: открыть на телефоне, отправить тестовую заявку (активирует Formspree), сгенерировать QR на полученный URL.
